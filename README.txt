# 📱 Mobile Tracking System (Java Project)

## 📌 Overview

The **Mobile Tracking System** is a simple Java-based desktop application built using **AWT (Abstract Window Toolkit)**. It allows users to:

* Track a mobile number (simulated location)
* Report a mobile as lost
* Display stored tracking and lost mobile data

This project is designed for academic purposes to demonstrate Java GUI concepts and event handling.

---

## 🚀 Features

* 📍 Track mobile number with simulated location
* 🚨 Report lost mobile numbers
* 📊 Display stored tracked and lost mobile data
* ❌ Exit confirmation dialog
* ✔ Input validation (10-digit mobile number)

---

## 🛠 Technologies Used

* Java
* AWT (GUI)
* Event Handling
* Collections (ArrayList)

---

## 📸 Application Functions

### 1. Track Mobile Location

* Enter a valid 10-digit number
* Displays simulated latitude & longitude

### 2. Report Lost Mobile

* Stores the number as lost
* Shows confirmation message

### 3. Display Stored Data

* Shows:

  * Tracked numbers
  * Lost mobile reports

---

## ⚠ Limitations

* Location tracking is simulated using random values
* No real GPS or database integration
* Data is not stored permanently
